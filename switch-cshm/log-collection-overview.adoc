= Log collection overview 
:icons: font
:imagesdir: ../media/

[.lead]
With log collection set up, you can enable an hourly collection of periodic data that is collected by AutoSupport, and request detailed support logs.

Log collection helps you troubleshoot network switch issues by automatically gathering switch logs and diagnostic data. You can collect periodic data on a scheduled basis through AutoSupport or trigger on-demand collection when you need detailed logs for troubleshooting specific issues. The collected logs provide visibility into switch health, configuration changes, and network events.

Refer to link:config-log-collection.html[Configure log collection] for further details.
