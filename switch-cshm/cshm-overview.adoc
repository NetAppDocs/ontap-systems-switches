= Ethernet switch health monitor
:icons: font
:imagesdir: ../media/

[.lead]
The Ethernet switch health monitor (CSHM) is responsible for ensuring the operational health of network switches and collecting switch logs for debugging purposes. 

CSHM monitors cluster and storage network switches to maintain network infrastructure reliability. The health monitor automatically detects switch issues, collects logs for troubleshooting, and provides alerts when network problems occur. CSHM supports multiple switch vendors and models, helping you identify and resolve network connectivity issues before they affect your storage environment.
