= Install the hardware for the NVIDIA SN2100 switch
:icons: font
:imagesdir: ../media/

[.lead]
Installing the NVIDIA SN2100 switch hardware is a critical first step in building a reliable and scalable network infrastructure to support data protection, replication, and management workflows.

Follow these steps to set up and configure your SN2100 switches. Refer to NVIDIA's documentation for detailed instructions.

.Steps

. Review the link:configure-reqs-sn2100-storage.html[configuration requirements].
. Follow the instructions in the link:https://docs.nvidia.com/networking/display/sn2000pub/Installation[NVIDIA Switch Installation Guide^].

.What's next?

After you've installed your hardware, you can link:cabling-considerations-sn2100-storage.html[review cabling and configuration] requirements.
