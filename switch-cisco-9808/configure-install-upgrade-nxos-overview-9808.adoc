= Install or upgrade the NX-OS overview - Cisco Nexus 9808
:icons: font
:imagesdir: ../media/

[.lead]
You can install or upgrade the NX-OS on the Nexus 9808 switches for AFX 2K systems. 

NOTE: NX-OS installation or upgrade is independent of RCF installation or upgrade. If you are installing or upgrading both the NX-OS and RCF, it is recommended you install or upgrade NX-OS first. 


* link:configure-install-nxos-9808.html[Install the NX-OS]
+
You install the NX-OS on the switches as part of a new AFX 2K system installation, a supervisor module replacement, or a switch reset to factory default settings.

* link:configure-upgrade-nxos-9808.html[Upgrade the NX-OS]
+
You upgrade the NX-OS on the switches to support new hardware or a new ONTAP version on your AFX 2K system.
