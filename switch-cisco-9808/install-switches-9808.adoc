= Install the switches - Cisco Nexus 9808
:icons: font
:imagesdir: ../media/

[.lead]
Installing the Cisco Nexus 9808 switches as part of a new AFX 2K system installation includes installing the switches into a rack or cabinet, cabling them to the system controllers and shelves, and then powering them on.

.Before you begin
You must have completed the switch preparation steps in link:install-prepare-9808.html[Prepare to install the switches].


.About this task

You use the applicable AFX 2K system install and setup workflow on the AFX system documentation site to install and cable the system hardware, and then power on the switches. 

The system install and setup workflow includes instructions such as:

* Instructions for getting the site ready, unpacking the boxes and comparing the contents of the boxes to the packing slip, and registering the system to access support benefits.

* Instructions for installing the switches, controllers, and storage shelves in a NetApp cabinet or telco rack.

* Instructions for cabling your system, including cabling the controllers to switch connections and cabling the storage shelves to switch connections.

.Step

. Install the switches as part of your AFX 2K system installation and setup workflow, go to the link:https://docs.netapp.com/us-en/ontap-afx/install-afx-2k/install-setup-workflow.html[AFX 2K install and setup workflow^].

NOTE: After you have installed the switches, the AFX install and setup workflow instructions redirect you back to this switch documentation to configure the switches.


.What's next?

After you've installed, cabled, and powered on your switches as part of your AFX 2K system installation, you link:install-configure-switches-in-afx-workflow-9808.html#configure-switches[configure your switches] and then link:install-configure-switches-in-afx-workflow-9808.html#complete-afx-system-setup[complete the AFX system setup].
