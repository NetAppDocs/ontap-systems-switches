= Install or update the NX-OS and RCF workflows - Cisco Nexus 9332D-GX2B
:icons: font
:imagesdir: ../media/

[.lead]
Lisa: do you still need this since you went back to old structure for NX-OS and RCF?

You install the NX-OS and RCF on the Cisco 9332D-GX2B switches as part of a new AFX system installation, a switch replacement, or a reset to factory default settings. You can also upgrade the NX-OS and/or RCF on an existing switch when you need a current version of either or both.

.Before you begin
If you are installing the NX-OS and RCF on switches as part of a new AFX system installation, a switch replacement, or a reset of the factory default settings, you must have link:setup-switch-9332d-cluster.html[completed the initial switch setup].

.About this task
* The difference between install and upgrade of the NX-OS and RCF is that for an install, you need to install both the NX-OS and RCF, whereas for an upgrade, you only install the NX-OS and/or RCF as needed.

* You can install or upgrade the NX-OS and RCF on the switch(es) in the AFX system one at a time, or on both switches at the same time. (Is this true? Does the procedure reflect this?)


.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png[One]link:install-nxos-overview-9332d-cluster.html[Prepare to install the NX-OS and RCF] 
[role="quick-margin-para"]
Prepare to install the NX-OS and RCF by gathering the necessary information and resources, such as the NX-OS image, RCF version, and switch information. You also need to determine the installation method you want to use for the NX-OS and RCF. (You mean download then install)

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png[Two]link:install-nxos-software-9332d-cluster.html[Install or upgrade the NX-OS]
[role="quick-margin-para"]
If this is part of an install, then install the NX-OS on the switch(es). If this is part of an upgrade, install the NX-OS if needed.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png[Three]link:install-upgrade-rcf-overview-cluster.html[Install or upgrade the RCF] 
[role="quick-margin-para"]
If this is part of an install, then install the RCF on the switch(es). If this is part of an upgrade, install the RCF if needed.


.What's next?

After you have installed or updated the NX-OS and/or RCF, you link:configure-ssh-keys.html[verify SSH is enabled].
