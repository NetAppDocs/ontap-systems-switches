= Installation and setup workflow - Cisco Nexus 9332D-GX2B

:icons: font
:imagesdir: ../media/

[.lead]
To install and set up your Cisco Nexus 9332D-GX2B switches as part of your new AFX storage system, you must prepare for the installation, install the switches, configure the switches, prepare to install the NX-OS software and RCF, Install the NX-OS software and RCF, configure switch health monitoring, and optionally, configure Smart Call Home.

Cisco Nexus 9332D-GX2B switches allow you to build ONTAP clusters with more than two nodes.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png[One]link:install-hardware.html[Prepare for install]
[role="quick-margin-para"]
Prepare to install and setup the switches by reviewing the Cisco and ONTAP systems documentation, preparing your site, and understanding the switch port usage.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png[Two]link:install-hardware.html[Install the hardware]
[role="quick-margin-para"]
You are directed to the AFX system installation and setup instructions to install the switches (along with the AFX storage system), and then you power on the switches. You are then directed back to this workflow to complete your switch setup.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png[Three]link:setup-switch-9332d-cluster.html[Configure the switches]
[role="quick-margin-para"]
Configure the switches. After you power on the switches, they begin to boot and enters the configuration wizard. You are prompted to provide answers to a series of questions to perform the initial (startup) configuration of the switches.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png[Four]link:install-nxos-overview-9332d-cluster.html[Prepare to install the software and RCF]
[role="quick-margin-para"]
Prepare to install the NX-OS software and RCF on the switches.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png[Five]link:configure-software-overview-9332d-cluster.html[Install the software]
[role="quick-margin-para"]
Install the NX-OS software on the switches.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-6.png[Six]link:configure-software-overview-9332d-cluster.html[Install the RCF]
[role="quick-margin-para"]
Install the RCF on the switches.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-7.png[Seven]link:configure-software-overview-9332d-cluster.html[Verify SSH is enabled]
[role="quick-margin-para"]
Verify SSH is enabled so that you can remotely manage the switches...configure health monitor.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-8.png[Eight]link:../switch-cshm/cshm-overview.html[Configure switch health monitoring]
[role="quick-margin-para"]
Configure the switch health monitoring (CSHM) to ensure the operational health of your switches and for the collection of switch logs for debugging and troubleshooting.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-9.png[Nine]link:smart-call-9332d-cluster.html[Configure Smart Call Home (optional)]
[role="quick-margin-para"]
Configuring Smart Call Home on your switches is optional. It is used to monitor the hardware and software components on your network. 

.What's next?

After you've installed and set up your switches as part of your AFX system installation, you complete the set up of your AFX system by link:/ontap-afx/install-setup/power-on-hardware.html[powering on your AFX 1K storage system].
