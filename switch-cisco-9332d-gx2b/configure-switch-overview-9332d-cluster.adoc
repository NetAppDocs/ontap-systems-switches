= Installation and setup workflow for Cisco Nexus 9332D-GX2B switches

:icons: font
:imagesdir: ../media/

[.lead]
To install and set up your Cisco Nexus 9332D-GX2B switches as part of your new AFX storage system, you must review the reference documentation, prepare your site, install and cable the hardware components, configure the switch software and RCF, configure switch health monitoring, and optionally, configure Smart Call Home.

Cisco Nexus 9332D-GX2B switches allow you to build ONTAP clusters with more than two nodes.

[role="quick-margin-para"]
Configure the switch health monitoring (CSHM) to ensure the operational health of your switches and for the collection of switch logs for debugging and troubleshooting.

[role="quick-margin-para"]
Configuring Smart Call Home is optional. It is used to monitor the hardware and software components on your network.
