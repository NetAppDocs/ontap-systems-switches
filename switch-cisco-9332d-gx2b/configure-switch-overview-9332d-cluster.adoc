= Install switches workflow - Cisco Nexus 9332D-GX2B

:icons: font
:imagesdir: ../media/

[.lead]
To install your Cisco Nexus 9332D-GX2B switches as part of your new AFX system, you must prepare for the installation and then install the switches (along with the AFX system hardware), which includes powering on and cabling the switches.

//.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png[One]link:configure-setup-ports-9332d.html[Review the switch port details]
//[role="quick-margin-para"]
//Review the port details for the Cisco 9332D-GX2B switch.

//.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png[One]link:required-documentation-9332d-cluster.html[Review the reference documentation]
//[role="quick-margin-para"]
//Review specific Cisco switch documentation and ONTAP systems documentation prepare for the install and setup of your Cisco Nexus 9332D-GX2B switches and AFX storage system.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png[One]link:install-hardware.html[Prepare for install]
[role="quick-margin-para"]
Prepare to install and setup the switches by reviewing the Cisco and ONTAP systems documentation, preparing your site, and understanding the switch port usage.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png[Two]link:install-hardware.html[Install the hardware]
[role="quick-margin-para"]
You are directed to the AFX system installation and setup instructions to install the switches (along with the AFX system hardware), cabling the switches, and then power on the switches. You are then directed back to this workflow to complete your switch setup (before completing your AFX system setup).

.What's next?

After you've installed, cabled, and powered on your switches as part of your AFX system installation, you link:setup-switch-workflow-9332d-cluster.html[set up your switches].
