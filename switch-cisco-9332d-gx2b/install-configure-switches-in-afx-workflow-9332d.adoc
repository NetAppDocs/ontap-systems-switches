= Install and configure switches in an AFX system workflow - Cisco Nexus 9332D-GX2B

:icons: font
:imagesdir: ../media/

[.lead]
To install and configure your Cisco Nexus 9332D-GX2B switches as part of your new AFX 1K or AFX 2K system, you use a combination of documentation from this switch site and the AFX systems site. The documentation provides links that direct you as needed.

=== Install switches
To install your switches, you must prepare for the installation and then install the switches (along with the AFX system hardware), which includes cabling and powering on the switches.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png[One]link:install-prepare-9332d-cluster.html[Prepare for install]
[role="quick-margin-para"]
Prepare to install and setup the switches by reviewing the Cisco and ONTAP systems documentation, preparing your site, and understanding the preset configuration of switch ports for the version of RCF supported by your AFX system.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png[Two]link:install-hardware.html[Install the switches]
[role="quick-margin-para"]
You are directed to the AFX 1K or AFX 2K system installation and setup workflow to install the switches (along with the AFX system hardware), cable the switches, and then power on the switches. You are then directed back to this workflow to configure the switches.

=== Configure switches
To configure your switches, you complete the initial setup, prepare to install NX-OS and RCF, install NX-OS, install RCF, verify SSH is enabled, configure Ethernet switch health monitoring (CSHM), and configure Smart Call Home (optional).

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png[One]link:setup-switch-9332d-cluster.html[Complete initial switch setup]
[role="quick-margin-para"]
After you power on the switches, they begin to boot and enter the configuration wizard. You are prompted to provide answers to a series of questions to perform the initial (startup) configuration of the switches.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png[Two]link:install-nxos-overview-9332d-cluster.html[Prepare to install NX-OS and RCF]
[role="quick-margin-para"]
Prepare to install NX-OS and RCF on the switches.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png[Three]link:configure-review-cabling-config-requirements-9332d.html[Review cabling and configuration requirements]
[role="quick-margin-para"]
Review cabling and configuration requirements.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png[Four]link:install-nxos-software-9332d-cluster-test.html[Install or upgrade NX-OS]
[role="quick-margin-para"]
Install the current version of NX-OS on the switches that supports your AFX 1K or AFX 2K system.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png[Five]link:install-upgrade-rcf-overview-cluster.html[Install the RCF]
[role="quick-margin-para"]
Install the version of RCF on the switches that supports your AFX 1K or AFX 2K system.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-6.png[Six]link:configure-verify-ssh-9332d.html[Verify SSH for log collection]
[role="quick-margin-para"]
Verify SSH is enabled. SSH is required for the  log collection feature to operate.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-7.png[Seven]link:../switch-cshm/cshm-overview.html[Configure Ethernet switch health monitoring]
[role="quick-margin-para"]
Configure the Ethernet switch health monitoring (CSHM) to ensure the operational health of your switches and for the collection of switch logs for debugging and troubleshooting.


.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-8.png[Eight]link:smart-call-9332d-cluster.html[Configure Smart Call Home (optional)]
[role="quick-margin-para"]
ONTAP supports Ethernet switch health monitoring (CSHM) to monitor the AFX cluster switches, as well as the ability to collect switch diagnostic logs. Cisco's Smart Call Home is an option to provide proactive support and help you resolve any issues that might arise with the switches.


=== Complete AFX system setup
To complete your AFX 1K or AFX 2K system setup, you complete the power up of AFX system components and system setup procedures, and then verify switch connectivity.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png[One]link:install-complete-afx-setup-9332d.html[Complete the AFX system setup]
[role="quick-margin-para"]
You are directed to the AFX systems documentation to complete the setup of your AFX 1K or AFX 2K system by powering on the storage shelves and controllers, and then completing the system setup procedures.

NOTE: After you complete the system setup, return to this topic to verify your switch connectivity to each other and to your AFX system. 



.What's next?

After you've completed the installation and configuration of your switches and AFX 1K or AFX 2K system, you can begin using your AFX system. You can also link:https://docs.netapp.com/us-en/ontap-afx/index.html[learn more about your AFX system and how to use it^].
